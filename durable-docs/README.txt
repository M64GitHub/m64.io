@synadia-ai/durable — documentation site
=========================================

A self-contained static site. To view it:

  → Open  index.html  in any web browser.
    (Double-click it, or drag it onto a browser window.)

No server, build step, or install needed — it's just HTML + CSS.

web font (it falls back to a system font if you're offline).
